# Lango
## Web-based Spanish-to-English translator and flashcard study tool.
*This is an archive of a previously completed university project.*

#### Uses
- JavaScript with React
- Node.js server
- Google Translator API
- SQL database

#### Other Information
"server1.js"	: This will run the server
		: server-URL = http://server162.site:50491/

"dump.js"	: This will print the contents of flash.db
"flash.db"	: database file that holds 'flashcards' and 'users'
"database"	: folder that holds the database-creation javascript
